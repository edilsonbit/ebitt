# Edilson Bittencourt — Personal Landing Page

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-6366f1?style=flat-square&logo=github)](https://edilsonbit.github.io/ebitt/)

Landing page pessoal desenvolvida em HTML, CSS e JavaScript puro, sem dependências de frameworks. O projeto transforma o currículo profissional em uma experiência visual imersiva com animações, efeitos de partículas e design moderno.

## ✨ Características

- Design escuro e futurista com paleta em indigo/violeta
- Animações de entrada via AOS (Animate On Scroll)
- Efeito de partículas interativas no hero (canvas)
- Typing effect no título principal
- Cards com glassmorphism e hover effects
- Timeline de experiência profissional
- Grade de competências técnicas com ícones
- Seção de premiações e formação
- Totalmente responsivo (mobile-first)
- Scroll suave e navbar com blur ao rolar

## 🛠 Tecnologias

- HTML5 semântico
- CSS3 (variáveis, grid, flexbox, animações)
- JavaScript vanilla (ES6+)
- [Space Grotesk](https://fonts.google.com/specimen/Space+Grotesk) + [Inter](https://fonts.google.com/specimen/Inter) — Google Fonts
- [AOS](https://michalsnik.github.io/aos/) — Animate On Scroll

## 🚀 Como usar

Basta abrir o `index.html` diretamente no navegador ou acessar via GitHub Pages:

```
https://edilsonbit.github.io/ebitt/
```
